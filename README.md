# To-Do List App

## Overview

This To-Do List app is designed to help users efficiently manage their tasks with a user-friendly interface. Utilizing local storage, the app allows for seamless task management, ensuring that your tasks are saved even after closing the browser.

## Features

- **Add, Edit, and Remove Tasks**: Easily create new tasks, modify existing ones, or delete tasks that are no longer needed.
- **Customizable Categories**: Each task can be assigned a category label, allowing for better organization and prioritization.
- **Priority Scale**: Tasks can be assigned a priority level, which is visually represented by a corresponding color, making it easy to identify urgent tasks at a glance.
- **Date and Time Management**: Users can set specific dates and times for tasks, helping to keep track of deadlines.
- **Task Status**: Mark tasks as complete or delete them as needed. Users can also view all tasks in one place for better oversight.

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Storage**: Local Storage for persistent data management

## Usage

1. **Add a Task**: Enter the task details, select a category, set a priority, and choose a date and time.
2. **Edit a Task**: Click on an existing task to modify its details.
3. **Remove a Task**: Delete tasks that are no longer relevant.
4. **Check Tasks**: Mark tasks as complete to keep track of your progress.
5. **View All Tasks**: Access a comprehensive list of all tasks for easy management.

This app aims to enhance productivity by providing a simple yet effective way to manage daily tasks.
